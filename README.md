# 🍩 Las Donas de Alme

Bienvenido/a al proyecto **Las Donas de Alme**, una página web dedicada 
a mostrar las mejores recetas y productos de nuestra donería.

## 🌐 Tecnologías usadas
- HTML
- JavaScript

## 📋 Contenido
- Recetas de donas artesanales
- Menú de productos disponibles

## 🚀 Cómo usar
1. Descargá el archivo `.zip`
2. Descomprimilo en tu computadora
3. Abrí el archivo `index.html` en tu navegador
